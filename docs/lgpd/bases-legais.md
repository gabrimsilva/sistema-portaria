## 3. Fundamentos Legais Detalhados

### 3.1 Execução de Contrato (Art. 7º, II)
**Aplicável a:**
- Funcionários (relação empregatícia) - dados gerais
- Prestadores de serviço (contratos de prestação) - dados gerais
- Usuários do sistema (relação trabalhista) - dados gerais

**Justificativa:** O tratamento é necessário para execução de contrato ou procedimentos preliminares relacionados a contrato do qual seja parte o titular, a pedido do titular dos dados.

### 3.2 Legítimo Interesse (Art. 7º, VI)
**Aplicável a:**
- Visitantes - dados gerais

**Justificativa:** O legítimo interesse é fundamentado na necessidade de:
- Garantir a segurança das instalações
- Proteger o patrimônio da empresa
- Cumprir obrigações de segurança no trabalho
- Controlar o acesso de pessoas não autorizadas

**Avaliação de Balanceamento:**
- Interesse legítimo da empresa em manter a segurança
- Dados mínimos necessários para identificação
- Tratamento proporcional e não excessivo
- Direitos dos titulares preservados

### 3.3 Dados Sensíveis - Prevenção à Fraude e Segurança (Art. 11, II, 'g')
**Aplicável a:**
- **TODOS os dados biométricos (fotos faciais)** para funcionários, visitantes e prestadores

**Justificativa Legal:** O Art. 11, II, 'g' da LGPD permite o tratamento de dados pessoais sensíveis para "prevenção à fraude e à segurança, em processos de identificação e autenticação de registro".

**Fundamentos Específicos:**
- **Prevenção à Fraude:** Evitar uso de identidades falsas ou documentos fraudulentos
- **Segurança das Instalações:** Garantir que apenas pessoas autorizadas tenham acesso
- **Identificação Precisa:** Assegurar correspondência entre a pessoa física e os dados cadastrais
- **Autenticação de Registro:** Validar a identidade no momento do acesso

**Proporcionalidade e Necessidade:**
- **Dados Mínimos:** Apenas foto facial, sem outros dados biométricos
- **Finalidade Específica:** Exclusivamente para controle de acesso e segurança
- **Não Excessivo:** Proporcional ao risco e necessidade de segurança
- **Medidas Técnicas:** Criptografia, acesso restrito, retenção limitada

**Alternativas Avaliadas:**
- **Consentimento:** Inadequado para acesso obrigatório (não seria livre)
- **Cumprimento de Obrigação Legal:** Não há norma específica exigindo dados biométricos
- **Legítimo Interesse:** Não aplicável para dados sensíveis conforme Art. 11

**Conclusão:** Art. 11, II, 'g' é a base legal mais adequada e específica para o tratamento de fotos faciais no contexto de controle de acesso empresarial.
